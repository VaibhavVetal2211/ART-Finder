PICT B.E. Course Structure

First Year (FE)

Core Subjects

  • Engineering Mathematics

    Advanced mathematical concepts and their applications in engineering

  • Engineering Physics

    Fundamental physics principles and their engineering applications

  • Engineering Chemistry

    Chemical principles and their applications in engineering

  • Basic Electronics

    Introduction to electronic components and circuits

  • Programming Fundamentals

    Introduction to programming concepts and problem-solving

Second Year (SE)

Core Subjects

  • Data Structures

    Study of fundamental data structures and algorithms

  • Object-Oriented Programming

    Advanced programming concepts and OOP principles

  • Database Management Systems

    Database design and management concepts

  • Computer Networks

    Network protocols and communication systems

  • Operating Systems

    OS concepts and system programming

Third Year (TE)

Core Subjects

  • Software Engineering

    Software development methodologies and practices

  • Web Technologies

    Modern web development frameworks and tools

  • Machine Learning

    Introduction to ML algorithms and applications

  • Cloud Computing

    Cloud platforms and distributed systems

  • Information Security

    Cybersecurity principles and practices

Final Year (BE)

Core Subjects

  • Project Work

    Capstone project implementation and documentation

  • Professional Ethics

    Ethical considerations in engineering practice

  • Advanced Topics

    Specialized subjects based on chosen track

  • Industry Internship

    Practical industry experience and training
